mac+hexo+github搭建博客

#mac+hexo+github+Markdown，搭建博客
一直以来，自己不断更换博客。用过的博客有网易，新浪，csdn，博客园，甚至还用过qq空间，还用过有道云笔记，微软onenote等等。每次迁移都是因为对自己的博客不满意，感觉自己的博客质量太差。有的博客内容仅仅是学习重点摘要，有的博客文章全篇都是摘录别人的，有的文章全是代码.

-

######因此给自己定下规矩：
1.在这个博客中，禁止大量摘录别人的东西，甚至全篇抄袭，可以引用别人的网址。

2.必须所有的东西都必须原创，将那些杂七杂八的文章都写在其他博客上。

3.每次想好开一个系列，必须把这个系列写完。

4.文章必须自己看着舒服，别人看着也舒服，自己若干年后再看自己的博客还会很开心。

5.每周必须写一篇博客。

6.这次博客必须系统化,不能东一榔头，西一榔头地写。这点非常重要，决定了自己会不会喜欢自己的博客。

7.这个博客的内容不在于多，而在于精。

8.博客的技术最好都引用官方博客，即使看别人写的也要注意看他从哪里引来的。

9.记录自己的学习过程。

-

######配置安装开始之前：
安装hexo以及配置github花了2天的时间，现在我把我按照官方的方法走的过程记录下。仅作参考，因为随着hexo版本的更新，一些命令之类的可能会变。

######安装任何环境及软件的基本方法:

第一反应按照官方的的步骤走。官方，官方，官方，重要的事说三遍。有的需要自备梯子，我用的vpn是免费的GreenVPN(官网)。

第二若果发现还是下载不下来的话，就看看有没有国内的镜像。国内网站记得看版本是不是最新的。

第三终端权限不够加sudo。

第三如果这条路还走不下去，就试着重新换种方法，环境，或者放弃重新选条路。

欢迎使用Markdown编辑器写博客

本Markdown编辑器使用StackEdit修改而来，用它写博客，将会带来全新的体验哦：

  • Markdown和扩展Markdown简洁的语法
  • 代码块高亮
  • 图片链接和图片上传
  • LaTex数学公式
  • UML序列图和流程图
  • 离线写博客
  • 导入导出Markdown文件
  • 丰富的快捷键

快捷键

  • 加粗 Ctrl + B
  • 斜体 Ctrl + I
  • 引用 Ctrl + Q
  • 插入链接 Ctrl + L
  • 插入代码 Ctrl + K
  • 插入图片 Ctrl + G
  • 提升标题 Ctrl + H
  • 有序列表 Ctrl + O
  • 无序列表 Ctrl + U
  • 横线 Ctrl + R
  • 撤销 Ctrl + Z
  • 重做 Ctrl + Y

Markdown及扩展

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成格式丰富的HTML页面。 —— [ 维基百科 ]

离线写博客

即使用户在没有网络的情况下，也可以通过本编辑器离线写博客（直接在曾经使用过的浏览器中输入write.blog.csdn.net/mdeditor即可。Markdown编辑器使用浏览器离线存储将内容保存在本地。

用户写博客的过程中，内容实时保存在浏览器缓存中，在用户关闭浏览器或者其它异常情况下，内容不会丢失。用户再次打开浏览器时，会显示上次用户正在编辑的没有发表的内容。

博客发表后，本地缓存将被删除。　

用户可以选择 把正在写的博客保存到服务器草稿箱，即使换浏览器或者清除缓存，内容也不会丢失。

注意：虽然浏览器存储大部分时候都比较可靠，但为了您的数据安全，在联网后，请务必及时发表或者保存到服务器草稿箱。
